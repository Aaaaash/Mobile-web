
#CSS3-Media Query <br/>
常见属性：<br/>
·device-width,device-height             ————————屏幕宽高<br/>
·width,height                           ————————渲染窗口宽高<br/>
·orientaion                             ————————设备方向<br/>
·resolution                             ————————设备分辨率<br/>

基本语法：<br/>
@media screen and (min-width:480px){
    body{background: blue}
}
